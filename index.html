UIUX 專案 與我聯繫
大學時參加 UI/UX 工作坊後，我開始關注產品背後的設計邏輯與使用者體驗，並持續觀察設計如何影響行為與決策。

現任 TinTint 點點印 UI/UX 設計師，擅長從用戶回饋中找出結構問題，提出具邏輯的優化方案，曾針對不同使用情境重構 Single、Editor2 編輯器與官網多個核心頁面，優化整體品牌介面體驗。我習慣從資訊架構出發，盤點功能、釐清邏輯，並與團隊協作推進設計落地。未來希望挑戰更多產品類型，打造兼具邏輯與美感的使用體驗。

Editor2｜作品線上編輯器
整體優化：打造更直覺流暢的編輯體驗

作為點點印全新一代的線上編輯器，Editor2 支援用戶在瀏覽器中輕鬆製作相片書、邀請卡、婚禮書約等個人化商品。透過拖拉照片、編輯文字與自訂版型，使用者能快速完成設計並直接下單印製。 我負責此產品的整體 UI/UX 設計，從編輯介面到操作流程，著重提升使用的直覺性與編輯彈性，並優化跨裝置體驗。最終成功讓用戶編輯更順暢、創作更自由，並有效提升作品完成率與轉換率。

Editor2 專案
Single｜輕量作品編輯器
聚焦效率與易用性，重新定義快速製作流程

為因應不同產品特性，Single 編輯器主打「簡單快速、直覺上手」的編輯體驗。相較於 Editor2 著重排版彈性與自訂功能，Single 聚焦極簡流程與自動排版，協助用戶在短時間內完成一本相片書設計。 我主導 Single 的整體 UI/UX Redesign，著重於操作動線優化、頁面引導簡化，並強化跨裝置的一致性與流暢度，讓更多用戶能快速上手，進一步提升作品完成率與下單轉換率。

TinTint Projects｜UI Enhancement
點點印品牌官網 多頁面的 UI 優化專案

為了提升整體網站的使用體驗與介面一致性，我參與了官網多個核心頁面的 UI 優化設計，包括「我的作品、作品列表、帳號設定頁、月曆系列產品頁、周邊配件頁與全產品一覽頁」...等。這些頁面多與會員操作與購買決策相關，設計上需兼顧資訊清晰、視覺引導與轉換效率。 我負責各頁介面的改版規劃與 UI 元件調整，並整合使用者行為觀察與內部回饋，優化操作流程與內容結構，使資訊更易讀、互動更直覺。此外，也針對既有設計進行視覺統一化與行動裝置友善調整，提升整體品牌一致性與跨裝置體驗。
